# 🎵 Lulu Suicide - Fan Site

> Um site não-oficial feito de fã para fã, homenageando a banda polonesa Lulu Suicide e sua estética dream-pop e melancólica (especialmente a era *Fiber Optic Lovers*).

🔗 **Live Demo:** [Visite o site aqui!](https://kaueotsubo.github.io/lulu-suicide-fansite/)

---

## ✨ Funcionalidades (Features)

* **Design Imersivo:** UI construída com *Glassmorphism* para criar uma atmosfera etérea que combina com o som da banda.
* **Animações em Canvas:** Um sistema de partículas desenvolvido em JavaScript puro (Vanilla JS) que roda no fundo da tela simulando uma poeira atmosférica/dream-pop.
* **Suporte Bilíngue (i18n):** O site possui um sistema de troca de idiomas instantâneo (Inglês e Polonês) sem necessidade de recarregar a página, manipulado via JS e atributos `data-i18n`.
* **Animações de Scroll:** Efeitos suaves de *fade-in* (*reveal*) ao rolar a página utilizando a API `IntersectionObserver`.
* **Totalmente Responsivo:** Layout adaptável para dispositivos móveis e desktops.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Semântica estruturada e atributos de dados personalizados.
* **CSS3:** Variáveis nativas (Custom Properties), Flexbox, CSS Grid, Backdrop-filter (efeito vidro) e animações de transição.
* **JavaScript (ES6):** Lógica do Canvas 2D, manipulação do DOM e observadores de interseção.
